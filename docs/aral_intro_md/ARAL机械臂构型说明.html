ARAL机械臂构型说明

ARAL机械臂构型介绍

1. 机械臂构型定义

ARAL 共定义了 8 种机械臂构型，其结构体定义如下：

enum class RobotConfiguration: int
{
    LEFT,           // 肩部朝左
    RIGHT,          // 肩部朝右
    UP,             // 肘部朝上
    DOWN,           // 肘部朝下
    FLIP,           // 腕部翻转
    NOFLIP,         // 腕部不翻转

    NONE    = -1,
    LUF     = 0,
    LUN     = 1,
    LDF     = 2,
    LDN     = 3,

    RUF     = 4,
    RUN     = 5,
    RDF     = 6,
    RDN     = 7
};
